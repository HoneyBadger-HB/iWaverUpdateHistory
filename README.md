## iWaver v.1.1.0 업데이트 내용

#### 1. PDF 이미지 선택 방법 변경
- Cmd(⌘)를 누르지 않아도 이미지를 다중 선택할 수 있습니다.
#### 2. 이미지 삭제 기능 단축키 추가
- Backspace를 통해 그리드에 추가한 이미지를 삭제할 수 있습니다.
#### 3. 작업 실행 취소, 다시 실행 기능 추가
- 작업 실행 취소(⌘+Z), 다시 실행(⌘+⇧+Z)의 단축키를 사용할 수 있습니다.
#### 4. 그 외 기타 버그를 수정하고 안정성을 강화했습니다.

## iWaver v.1.1.0 Update Notes

#### 1. Enhanced image selection in PDFs
- You can now select multiple images without holding down the Cmd(⌘) key.
#### 2. Image deletion with Backspace
- You can now delete images that you have added to the grid by pressing the Backspace key.
#### 3. Undo and Redo functionality
- Added support for undo (⌘+Z) and redo (⌘+⇧+Z) actions.
#### 4. Various bug fixes and stability improvements.
